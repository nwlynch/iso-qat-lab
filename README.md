# /qa-lab/ Root Directory Structure
This directory holds all scaffolding for the QA Lab.
## Agents
- /agents/: Contains the core logic modules for each agent.
## Tests
- /tests/: Contains standardized test files (unit, e2e, etc.).
## Results
- /results/: Stores all output artifacts (reports, logs, metrics).
## Config
- /config/: Stores reusable configuration files (model settings, framework versions).
## Docs
- /docs/: Stores high-level strategy documents and wikis.

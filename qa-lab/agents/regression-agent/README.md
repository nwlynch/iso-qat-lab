# 📋 Regression Agent

## Purpose
Automatically runs regression tests, tracks changes, and maintains test baseline.

## Responsibilities
- Run regression test suites
- Compare with baseline
- Identify breaking changes
- Assess impact of changes
- Generate regression reports

## Input
- Test suite
- Baseline results
- Code changes
- Environment state

## Output
- Regression results
- Breaking change reports
- Impact assessments
- Updated baseline
- Change logs

## Model
- Default: llama3.1:8b (efficient)
- Complex suites: qwen3.5:9b

## Workflow Integration
- Phase 6: Regression & Maintenance
- Feeds into Phase 7: Continuous Improvement
